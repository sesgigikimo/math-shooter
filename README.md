# 數學射擊大作戰

給小朋友玩的數學射擊小遊戲。怪物從天而降、腳下踩著算式；點對的答案，砲台才會發射電漿彈把怪物擊毀。

純單檔 HTML，無任何依賴，雙擊就能玩。

## 開始玩

```
直接用瀏覽器打開 index.html
```

或者 clone 下來：

```bash
git clone https://github.com/sesgigikimo/math-shooter.git
open math-shooter/index.html
```

## 遊戲規則

- 100 秒倒數，目標是答對最多題
- 怪物從畫面上方緩慢落下，題目永遠顯示在怪物腳下
- 下方有三個答案，**只有點對的才會發射砲彈**
- 答錯沒有懲罰，只是連擊歸零、按鈕震一下
- 怪物落到底也不會消失，會停下來等你回答
- 答對之後新怪物立刻出現，沒有空檔
- 最高紀錄自動存在瀏覽器（localStorage），下次打開還在

## 題型與難度

開始畫面可以**勾選多個題型混合出題**（加、減、乘、除）。

| 運算 | 簡單 | 普通 | 困難 |
| --- | --- | --- | --- |
| 加法 | 1-9 ＋ 1-9（個位數） | 和 ≤ 20 | 和 ≤ 100 |
| 減法 | 被減數 ≤ 9 | ≤ 20 | ≤ 100 |
| 乘法 | 1-5 × 1-5 | 1-5 × 1-9 | 1-9 × 1-9（九九乘法） |
| 除法 | 對應乘法的逆運算（整除） | 同左 | 同左 |

## 操作

- 滑鼠：點下方答案按鈕
- 鍵盤：`1` `2` `3` 對應三個答案
- 右上 🔊：靜音 / 開聲
- 右上 🏁 結束：直接結束本場、看成績
- 結算畫面：顯示本次答對題數、最高紀錄、最高連擊，破紀錄會跳出 🏆 徽章

## 技術

- 單檔 HTML / CSS / JavaScript
- HTML5 Canvas 繪製砲台、怪物、粒子、雷射
- Web Audio API 即時合成音效（不需音檔）
- localStorage 儲存最高紀錄
- 無建置流程、無套件依賴
